CULTURAL FASHIONOVA



This website is made for everybody. The goal is to give
cultural fashion a platform to showcase their worth. Basically, this website is like an e-commerce
website but has a particular range and style of
products.

The features of this website include:

  1. An e-commerce website
  2. Can learn the history behind the products
  3. Has multiple pages and animations on products
  4. Has a rating feature
  5. Provides and focuses with detail on each products
